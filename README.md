## はじめに

本プロジェクトは、S&P500の時系列データに基づき、深層学習を用いて未来の価格を予測するものです。


## 使用モデル
+ LSTM
+ GRU
+ CNN-LSTM
+ LSTM-ResNet
+ ResNet
+ Transformer
+ TIMESNET

## 使用データ
S&P500 日足終値データ
